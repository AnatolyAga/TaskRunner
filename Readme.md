# TaskRunner

**TaskRunner - это утилита для автоматической выгрузки и сохранении данных из СУБД Postgres**

## Основные функции: 
**Выгрузка по расписанию:** выгружает данные в заданное время

**Повторность выгрузок**: возможность повторной выгрузки через интервал заданный в минутах

**Отдельный SQL запрос для каждой задачи:** каждая задача выполняется со своим запросом

**Отдельный путь сохранения выгрузки из каждой задачи:** отдельный excel файл для каждой задачи

**Экспорт и импорт списка задач:** возможность сохранить созданные задачи для их переиспользования
